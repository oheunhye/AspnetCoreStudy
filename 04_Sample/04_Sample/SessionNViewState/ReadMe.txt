1 세션 상태 Session State
ASP.NET세션상태는 제한된 시간 창에서 동일한 브라우저의 요청을 세션으로 식별하며
해당 세션동안 변수 값을 유지하는 방법을 제공


-세션변수
하나의 클라이언트가 서버내의 여러 웹 페이지에서 데이터를 공유할 때 사용
ASP.NET페이지에서 현재 세션 변수는 Page개체의 Session속성을 통해 표시
세션 변수의 컬렉션은 변수 이름이나 정수 인덱스로 인덱싱
어플리케이션안에 세션이 포함되는 구조.

-어플리케이션과의 차이점
어플리케이션 :여러 사용자가 브라우저를 하나의 정보를 같이 공유가능
세션		 :한 사용자가 사용한 페이지들의 정보를 공유 ex)장바구니

-세션변수 사용방법
Session("세션변수명") = 값;

2 뷰 상태 ViewState
웹 서버와 웹브라우저 간에 데이터를 송수신하여 *페이지의 상태를 저장*하는 기능
웹 서버에 대한 라운드트립이 발생할 때마다 페이지 및 컨트롤 값을 유지할 수 있는 기능이 필요
웹서버가 여러번 요청을 받는 경우가 많기 때문에, 뷰스테이트로 요청정보,페이지정보를
페이지내에 숨김필드로 저장시켜 둔다.
렌더링이 빠르나 보안에 취약하고 정보가 많으면 통신트래픽이 오래 걸림.

-라운드트립 =포스트 백
		   ------요청------>
웹브라우저					웹서버
		   <------HTML------(처리하고 지움)
		   ------재요청------>
		   <------HTML------(포스트백)




-뷰상태 사용방법
페이지의 "ViewState"속성을 사용하여 코드에서 뷰 상태에 액세스 가능
ViewState의 정보는 System.Web.Ui.StateBag클래스에서 관리

-뷰상태 사용방법
ViewState["뷰스테이트명"] = 값;
